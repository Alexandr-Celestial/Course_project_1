# Проект: Обработка финансовых операций
Этот проект предназначен для обработки и анализа финансовых операций, хранящихся в Excel-файле. 
Включает в себя функции для поиска данных, фильтрации и анализа операций.
## Основные функции
### `read_excel_file`
Функция для чтения данных из Excel-файла. Возвращает список словарей, где каждый словарь представляет строку из файла.
### `rec_search_request`
Функция для поиска данных по ключевому слову. Ищет совпадения в полях "Категория" и "Описание".
### `search_mobile_number_excel`
Функция для поиска мобильных номеров в поле "Категория". 
Использует регулярные выражения для поиска номеров в формате `+7 XXX XX-XX-XX`.
### `search_transactions_to_people`
Функция для поиска переводов физическим лицам. Ищет операции в категории "Переводы" и проверяет формат имени получателя.
## Установка
1. Убедитесь, что у вас установлен Python 3.8 или выше.
2. Клонируйте репозиторий:
git clone https://github.com/Alexandr-Celestial/Course_project_1
3. Используйте функции для обработки данных.
## Примеры
### Поиск данных по ключевому слову
```
from src.services import rec_search_request
result = rec_search_request("Кафе")
print(result)
```
### Поиск мобильных номеров
```
from src.services import search_mobile_number_excel
result = search_mobile_number_excel()
print(result)
```
### Поиск переводов физическим лицам
```
from src.services import search_transactions_to_people
result = search_transactions_to_people()
print(result)
```
## Тестирование
Для тестирования проекта используется библиотека `pytest`. 
Тесты написаны для всех функций проекта и находятся в директории `tests/`.
Для запуска тестов выполните следующую команду:
```
pytest --cov=src --cov-report=html
```
Отчет о покрытии кода будет создан в папке Отчет будет сгенерирован в папке `htmlcov` 
и храниться в файле с названием `index.html`
## Логирование
Логи записываются в файл `logs/services.log`. Включает информацию о выполнении функций и возможных ошибках.
